# Web Projects by Vikas Singh

This repository contains three responsive HTML & CSS-based web development mini projects:

1. **🧮 Calculator App**
2. **🛍️ Tech Gadgets Landing Page**
3. **🍕 Food Ordering Website (Tasty Bites)**

Each project is fully responsive, styled using internal CSS, and interactive using JavaScript (where applicable).

---

## 📁 Project 1: Calculator App

### 📋 Description
A fully functional scientific-style calculator built using **HTML**, **CSS**, and **JavaScript**. It supports:
- Basic arithmetic operations (+, -, *, /)
- Decimal inputs
- Parentheses
- Keyboard input
- Backspace and Clear functionality

### 🖥️ Technologies
- HTML5
- CSS3 (Grid, Flexbox, Hover Effects)
- JavaScript (DOM, `eval()`, `keydown` events)

### ▶️ How to Use
1. Open `calculator.html` in a browser.
2. Use mouse or keyboard to enter expressions.
3. Press `=` or `Enter` to evaluate.
